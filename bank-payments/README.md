Programa turi sugebeti atspausdinti banku mokejimo israsu duomenys, kur:

* Datos formatas: YYYY-MM-DD
* Mokejimo sumos formatas: XXX.XX (su tasku)
* Ataskaitos forma: Eil. Nr., Data, Suma, Moketojas, Saskaitos Nr

Reikalavimai:
* Programa turi sugebeti patekti ataskaita (grazia forma) atskiro banko (pasirinktinai)
* Programa turi sugebeti pateikti ataskaita (grazia forma) visu bako mokejimu

Bonus:
* yra galimybe filtruoti pagal data/datos riezius
* sarasas rusiuojamas pagal data

Techniniai reikalavimai:
* OOP
* Clean Code (jokiu kodo dubliu)
* Programa turi buti parasyta taip, kad galima butu praplesti/prideti papildomus bankus
* For table view use <a href="https://www.asciitable.com/">ASCII Table</a>:
  * from 176 -> 223
